## Research workflow

When we do research together, publish an essential version of the findings to `.agents/research/` as simple markdown.

Each research note should contain:
- A distilled, essential version of the knowledge found (the core takeaways)
- References/links to the sources consulted

Keep the format simple — plain markdown, no heavy structure. Name files with a numeric prefix to preserve order: `.agents/research/01-topic.md`, `02-topic.md`, etc.
